## 함수

함수는 입력값을 받아 새로운 정보를 반환한다. 수학에서 함수를 정의하는 방법은 두 가지가 있다.

첫 번째는 참조 테이블을 만드는 방법이다:

| x | y |
|---|---|
| 0 | 0 |
| 1 | 2 |
| 2 | 4 |
| 3 | 6 |
| 4 | 8 |

하지만 이 방식은 테이블에 없는 값을 처리할 수 없고, 많은 케이스를 다루려면 거대한 테이블이 필요하다는 단점이 있다.

따라서 두 번째 방법으로 `y = 2x`와 같이 방정식으로 정의할 수 있다. 이 방식은 모든 x값에 대한 y값을 구할 수 있다.

## 좌표계

### 화면 좌표계 vs 데카르트 좌표계

화면 좌표계는 좌상단이 원점이며, 아래로 갈수록 y값이 증가한다. 반면 데카르트 좌표계는 좌하단이 원점이며, 위로 갈수록 y값이 증가한다.

![화면 좌표계와 데카르트 좌표계](/imgs/coord_system_2d.png)

그래픽스 프로그래밍에서는 데카르트 좌표계에서 작업한 후 최종 렌더링 시 화면 좌표계로 변환한다.

### 3차원 좌표계

![3차원 오른손 좌표계](/imgs/coord_system_3d.png)

오른손 좌표계에서는 z축이 관찰자 쪽으로 뻗어 나온다. OpenGL과 WebGL에서 사용하는 좌표계이다.

2차원에서 점은 `(x, y)`로, 3차원에서는 `(x, y, z)`로 표기한다.

## 직선의 정의

직선의 방정식은 다음과 같이 정의된다:

> **Ax + By = C**

예를 들어 `y = 2x`는 `2x - y = 0`으로 나타낼 수 있고, `y = 3`은 `0x + y = 3`으로 표현할 수 있다.

![y = 2x의 해집합](/imgs/y=2x.png)

## 직선의 성질

### 기울기

기울기는 상승량을 이동량으로 나눈 값이다:

> **기울기 m = (y₂ - y₁) / (x₂ - x₁)**

표준형 방정식 `Ax + By = C`에서:

> **m = -A/B**

기울기의 의미:

- 음수: 왼쪽에서 오른쪽으로 내려간다
- 양수: 왼쪽에서 오른쪽으로 올라간다
- 0: 수평선
- 정의 불가: 수직선

### 평행과 직교

- 두 직선의 기울기가 같으면 평행하다
- 두 직선이 직교할 때: **m₁m₂ = -1**

### 기울기-절편 공식

> **y = mx + b**

b는 y절편으로, 직선이 y축과 만나는 점이다(x = 0일 때의 y값).

### 점-기울기 공식

> **(y - y₁) = m(x - x₁)**

직선 위의 한 점과 기울기만으로 직선의 방정식을 구할 수 있다.

### 실제 예시

캐릭터가 S(50, 200)에서 E(150, 400)으로 이동할 때:

1. 기울기: m = (400 - 200) / (150 - 50) = 2
2. 점-기울기 공식 적용: (y - 400) = 2(x - 150)
3. 정리: y = 2x + 100

### 직교하는 직선 구하기

물체가 `y = 2/3x + 20`을 따라 이동하다가 (30, 40)에서 90도 꺾이는 경우:

1. 원래 기울기: m = 2/3
2. 직교하는 기울기: m₂ = -3/2
3. 점-기울기 공식: (y - 40) = -3/2(x - 30)
4. 결과: y = -3/2x + 85

![직교하는 두 직선](/imgs/ortho-1.png)

### 3차원 직선

3차원에서는 벡터 표기를 사용한다:

- 점: `<x, y, z>`
- 기울기(방향): `<Δx, Δy, Δz>`

예: 점 `<50, 200, 75>`에서 `<100, 50, 225>`로 이동

- Δx = 50, Δy = -150, Δz = 150
- 변위: `<50, -150, 150>`

## 직선의 성질을 이용한 충돌 검사

두 직선의 관계:

- 기울기가 다르면: 충돌지점 1개
- 기울기와 y절편이 모두 같으면: 충돌지점 무한개
- 기울기는 같으나 y절편이 다르면: 충돌지점 0개

## 두 직선의 교차점 구하기

### 선형결합법

예제:

```
3x + 2y = 10
4x + 3y = 6
```

1. y의 계수를 일치시키기 위해 양변에 공배수를 곱한다
2. 한 식에서 다른 식을 빼서 한 변수를 소거한다
3. 한 변수의 값을 구한다
4. 그 값을 원래 식에 대입하여 나머지 변수를 구한다

결과: (18, -22)

### 치환법

예제:

```
x + 2y = 5
3x - 2y = -1
```

1. 첫 번째 식을 x에 대해 정리: x = -2y + 5
2. 두 번째 식에 대입: 3(-2y + 5) - 2y = -1
3. y 구하기: y = 2
4. x 구하기: x = 1

결과: (1, 2)

### 일반 공식

두 직선이 교차할 때 교점의 x좌표:

> **x = (m₁x₁ - m₂x₂ + y₂ - y₁) / (m₁ - m₂)**

교점의 y좌표:

> **y = m₁(x - x₁) + y₁**

### JavaScript 코드

```javascript
/**
 * point1, point2는 x와 y를 가지는 좌표 객체
 * slope1, slope2는 기울기
 */
function getLineIntersect(point1, slope1, point2, slope2) {
    var intersectX = 0,
        intersectY = 0;

    // x = (m1x1 - m2x2 + y2 - y1) / (m1 - m2)
    intersectX =
        (slope1 * point1.x - slope2 * point2.x + point2.y - point1.y) / (slope1 - slope2);
    // y = m1(x - x1) + y1
    intersectY = slope1 * (intersectX - point1.x) + point1.y;

    return {
        x: intersectX,
        y: intersectY,
    };
}
```

## 결론

선형결합법과 치환법은 모두 1차 연립방정식을 풀 수 있다. 한 변수의 계수가 1이면 치환법이 유리하고, 그렇지 않으면 선형결합법을 사용하는 것이 좋다.
