John Allsopp(Web Directions)의 조사에 따르면, 2010년 웹개발 설문은 브라우저 사용, 기술 도입, 개발자 선호에서 의미 있는 변화를 보여주었다.

## 브라우저 동향

- IE는 개인 사용률이 낮음에도 불구하고 가장 많이 테스트되는 브라우저로 남았다
- Chrome이 빠르게 3위로 올라서며 Safari의 20%에 거의 근접한 17% 사용률을 기록했다
- Firefox는 1위를 유지했으며, 3.5와 3.6 버전 간 뚜렷한 차이를 보였다
- Mac OSX가 응답자들 사이에서 주류 운영체제로 부상했다

## 모바일 개발

- Mobile Safari 사용률이 16%에 도달 (전년 대비 증가)
- Android 도입은 4%로 미미한 수준이었다

## 프레임워크 채택률

- jQuery가 80%의 개발자 사용률로 압도적 (이전 63%에서 상승)
- Cappuccino, SproutCore 같은 데스크톱 애플리케이션 스타일 프레임워크는 제한적 채택

## 신기술 도입

- **CSS3:** 45% 도입 (전년 22%에서 상승)
- **Web fonts (@font-face):** 23% 도입 (전년 4%에서 상승)
- **HTML5:** 30%가 일부 기능 사용 (전년 10%에서 상승)

## 전망

- Mac 기반 개발의 증가
- IE 테스트 부담의 지속
- jQuery의 지속적 지배
- HTML5/CSS3의 더 넓은 도입
- PhoneGap, Appcelerator Titanium 같은 하이브리드 앱 프레임워크의 성장
